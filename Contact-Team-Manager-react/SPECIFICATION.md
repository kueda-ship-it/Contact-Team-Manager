# Contact Team Manager - 仕様書

## 概要
連絡先とチーム管理を行うためのウェブアプリケーションです。React と Vite を使用して構築されています。

## 技術スタック
- **Frontend**: React (v19), TypeScript
- **Build Tool**: Vite (v7)
- **CSS**: Vanilla CSS (PostCSS)
- **Backend/Services**: Supabase, Microsoft Graph API (MSAL)

## 対応プラットフォーム
- **Web Browser**: Chrome, Edge, Safari (Modern versions)
- **OS**: macOS, Windows, Mobile (iOS/Android)

## 主な機能
- **フィード/スレッド**: チームごとの連絡事項の投稿、返信、ステータス管理、メンション機能（@ユーザー/@タグ）。
  - **アクションアイコンの配色とホバーエフェクト**: 通常時は白、ホバー時にアクションに応じた色（水色、黄色、シアングリーン、紫）とスケルトングローが適用されるインタラクティブなデザイン。直感的な操作感を実現。
  - **URL自動リンクとプレビュー**: 投稿内のURLを自動的にリンク化し、Supabase Edge Function (`get-link-preview`) を利用してOGP情報（タイトル、説明、画像）を動的に取得・表示。
  - **画像アップロード即時プレビューと詳細確認**: 
    - 投稿・返信時にファイルを選択または画像を貼り付けた際、アップロード完了を待たずに即座にプレビューを表示。
    - プレビューサイズを拡大（最大 200x150）し、クリックでフルサイズ確認が可能。
- **データ分析**:
  - 返信の有無による解決スピードの比較分析。
  - 長時間スレッドランキング（Top 100）およびスレッドへのダイレクトナビゲーション。
- **ユーザー管理（ホワイトリスト）**: 管理者による事前メールアドレス登録制。初回ログイン時に自動的にプロフィールを生成。
  - **SSO 認証の柔軟性**: メールアドレスの照合を大文字小文字に関わらず実行（Case-insensitive）。Microsoft 側のアカウント表記に依存しない安定したログインを実現。
- **ナレッジ共有**: 内部ナレッジ機能（開発中）および外部リンク。

## 開発環境のセットアップ
1. `npm install`
2. `.env` ファイルに Supabase と Azure の接続情報を設定
3. `npm run dev` で開発サーバーを起動
